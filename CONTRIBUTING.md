# Contributing to GraviCore AI 🚀

Thanks for your interest in contributing!

## 🧠 How to Contribute

- Fork the repository
- Create a new branch
- Make your changes
- Submit a pull request

## 💡 Contribution Areas

- AI features (Ollama integration)
- UI/UX improvements
- Automation workflows
- 3D assistant (Three.js)

## ⚠️ Guidelines

- Keep code clean and readable
- Follow existing structure
- Test before submitting

## 🚀 Vision

We are building a next-generation AI Operating System. Contributions are welcome!
